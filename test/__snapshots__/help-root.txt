Command-line workflows for recurring jobs, local schedulers, and rrule.net cloud schedules.

VERSION
  <VERSION>

USAGE
  $ rrulenet [COMMAND]

TOPICS
  cloud   Manage rrule.net cloud schedules
  config  Manage rrulenet CLI configuration
  import  Import schedules from other formats
  local   Manage local recurring schedules

COMMANDS
  help      Display help for rrulenet
  list      List schedules from local/cloud/all
  simulate  Simulate future occurrences for an RRule
