Add a local schedule

USAGE
  $ rrulenet local add RRULE... [--json] [--timezone <value>]
    [--dtstart <value>] [--name <value>]

ARGUMENTS
  RRULE...  RRule expression

FLAGS
  --dtstart=<value>   ISO start datetime
  --name=<value>      Schedule name
  --timezone=<value>  [default: UTC] IANA timezone

GLOBAL FLAGS
  --json  Format output as json.

EXAMPLES
  $ rrulenet local add "FREQ=DAILY;BYHOUR=9;BYMINUTE=0;BYSECOND=0" -- echo "sync"

  $ rrulenet local add "FREQ=WEEKLY;BYDAY=MO;BYHOUR=18;BYMINUTE=0;BYSECOND=0" --timezone Europe/Paris --dtstart 2026-04-06T16:00:00.000Z --name "Monday deploy" -- ./deploy.sh
